# davidtaylorjr.github.io

Personal website for David Taylor, independent software developer and founder of Taylor Creative Development.

## Stack

- Semantic HTML
- Modern CSS
- GitHub Pages
- No framework, build step, or runtime dependencies

## Local preview

Open `index.html` in a browser or run a small static server from the repository root:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
